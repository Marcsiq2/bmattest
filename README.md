# bmattest
Bmat Test

## Usage
+ MySQL DB creation
```
usage: mysql -u <user> -p < create_db.sql
```
+ Server
```
usage: server.py [-h] [-H MYSQL_HOSTNAME] [-U MYSQL_USER] [-P MYSQL_PASSWORD]
                 [-D MYSQL_DATABASE]
```
+ Test
```
usage: test.py [-h] [-H HOSTNAME] [-P PORT] [--add-data]
```
